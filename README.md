## 🔘 Fields of EventoModel are:

+ nome
+ local

## 📔 Features
This API has only a feature that allows a event CRUD.

## 🤝🏽 Business Rules
Rotes PUT,DELETE and GET byID return event not found case id does not exist on the database.
